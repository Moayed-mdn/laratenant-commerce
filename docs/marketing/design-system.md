# Marketing Design System

## Visual Direction

- modern SaaS aesthetic
- clean whitespace
- low visual noise
- typography-first hierarchy
- subtle motion only

## Container Rules

- max width: 1280px
- marketing sections use consistent horizontal padding
- avoid arbitrary widths

## Typography

- hero heading:
  - text-5xl desktop
  - text-3xl mobile
  - font-bold
- section headings:
  - text-3xl
- body:
  - text-base
  - relaxed leading

## Buttons

Primary:
- solid brand color
- high contrast

Secondary:
- outline or ghost

## Motion

- prefer subtle fade/slide
- avoid parallax
- avoid excessive animation

## Sections

Preferred order:
1. Hero
2. Social proof
3. Features
4. Showcase
5. Testimonials
6. Pricing
7. FAQ
8. CTA
